# بوت النشر التلقائي
بوت تليجرام لإدارة النشر في المجموعات عبر الحسابات الشخصية.

## التثبيت
1. قم بتثبيت المكتبات:
   `pip install -r requirements.txt`
2. أضف متغيرات البيئة في النظام:
   - `API_ID`
   - `API_HASH`
   - `BOT_TOKEN`
3. شغل البوت:
   `python main.py`

## النشر على استضافة بايثون
يمكنك تشغيل البوت على أي استضافة بايثون تدعم تشغيل سكربتات طويلة (Long-running). الإعدادات العامة:
1. ارفع الملفات إلى الاستضافة.
2. ثبت المتطلبات:
   `pip install -r requirements.txt`
3. أضف متغيرات البيئة (`API_ID`, `API_HASH`, `BOT_TOKEN`) من لوحة التحكم.
4. اجعل أمر التشغيل:
   `python main.py`
